# Mobile Technical Task
Aplikasi *mobile* untuk menyarankan resep makan siang

# Libary 
- provider
- dio
- get_it
- json_annotation
- flutter_datetime_picker
- intl
- build_runner
- json_serializable
- mockito

# Run project
- Clone project nya
- Buka di IDE seperti VisualStudioCode, dll
- Coba jalankan, pastikan emulator atau device telah siap.
